# reviews/ — Individual Trade Reviews

One file per trade, generated by Fortuna after each session. Covers trade data, behavioral analysis, pattern tracking, and notes for coaches.

## 📂 Structure

```
reviews/
├── pattern_tracker.md          ← active behavioral pattern log
└── YYYY/
    └── MM-Mon/
        └── review_YYYYMMDD_[instrument]_NNN.md
```

*NNN = day-sequential number, resets each date (001, 002, 003...)*

## 📋 Review Sections (Standard)

1. ⚡ What Happened
2. 📊 Trade Data (table from CSV)
3. 🧠 Behavioral Notes
4. 📝 Notes for Coaches
5. 🔁 Pattern Tracker
6. 📋 Order Execution
7. 📸 Screenshot Timeline (full-size embeds)
