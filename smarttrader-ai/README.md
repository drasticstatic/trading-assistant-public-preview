# smarttrader-ai/ — Session Reviews & Coaching Exports

This directory holds all session analysis, trade reviews, pre-market summaries, and SmartTraderAI coaching exports generated by Fortuna (Claude Code CLI).

## 📂 Subdirectories

| Directory | Contents |
|-----------|----------|
| `analysis/premarket/` | Pre-market summaries — bias, levels, news, expectations |
| `reviews/` | Individual trade reviews — one file per trade, date-sequential numbering |
| `exports/` | SmartTraderAI copy-paste exports (daily + weekly formats) |

## 📄 File Naming Conventions

| File Type | Pattern | Example |
|-----------|---------|---------|
| Pre-market summary | `premarket_YYYYMMDD_summary.md` | `premarket_20260325_summary.md` |
| Per-instrument pre-market | `premarket_YYYYMMDD_[instrument].md` | `premarket_20260325_CL.md` |
| Trade review | `review_YYYYMMDD_[instrument]_NNN.md` | `review_20260316_RTY_002.md` |
| Daily STB export | `STB_export_YYYYMMDD_daily-review.md` | `STB_export_20260316_daily-review.md` |
| Weekly STB export | `STB_export_YYYYMMDD_weekly-review.md` | `STB_export_20260322_weekly-review.md` |

*NNN = day-sequential number (resets each date, not global)*
