# exports/ — SmartTraderAI Coaching Exports

Ready-to-paste coaching check-ins for the SmartTraderAI platform, generated by Fortuna from session data.

## 📂 Structure

```
exports/
└── YYYY/
    └── MM-Mon/
        ├── STB_export_YYYYMMDD_daily-review.md
        └── STB_export_YYYYMMDD_weekly-review.md
```

## 📋 Export Formats

### Daily Post-Market Review (3 Questions)
- Q1: What happened today?
- Q2: What did I learn?
- Q3: Results — P/L, rule adherence, grade

### Weekly Check-in (9 Questions + Action Steps)
- What worked / didn't work
- Observable patterns
- Mistakes + solutions
- Yes/No performance questions
- Action steps for the coming week
- Study plan
