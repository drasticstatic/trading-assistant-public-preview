- **Situational pressure acknowledged:** Financial stress + prop firm progression pressure is real context that shapes every trade decision. The nerves are not irrational — they're information. The question is whether the process can hold steady when the pressure is high.
- **ZTH 2R as the lesson:** The exact scenario Christopher watched — BE stop followed by re-entry on confirmation — is the process answer to both the hesitation and the fear. The second entry is only possible if the first was managed correctly (BE stop, not a loss). That's the chain reaction the process protects.

---

## 🔑 Key Lessons

1. **A limit order set too far away is still a no-trade.** Seeing the move and not participating has the same financial outcome as not watching at all — but costs more psychologically. If the read is there, the limit needs to be at the level, not 10 pts away from it.

2. **BE stops create second chances.** ZTH's 2R didn't happen on the first entry. It happened because the first entry was managed to BE, freeing capital and psychology for the re-entry. The process that feels like "losing" (stopping out at BE) is what made the win possible.

3. **Volatility is the game right now, not the exception.** CPI + tariff environment = fast moves, fast reversals. The coaches aren't fighting this — they're using it. Quick partials, immediate BE, then let the runner work. This is the operating mode until macro settles.

4. **Nerves are information, not disqualifiers.** Feeling nervous about the situation is honest self-awareness. The goal isn't to feel fearless — it's to execute the process anyway. Every correct process rep in a nervous state builds the muscle more than a calm-day trade does.
